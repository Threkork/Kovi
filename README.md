# Kovi

Kovi 是一个 OneBot V11 的插件框架，如果你想用 Rust 来开发 OneBot V11 机器人，那么 Kovi 是一个绝佳选择。

目标是 Rust 最简单 OneBot 框架！

复杂的 Rust 简化写法？ Kovi 已经尽力了。

本项目文档非常简单易懂，跟着来一遍保证就会了。

[Kovi Doc](https://threkork.github.io/kovi-doc/)
